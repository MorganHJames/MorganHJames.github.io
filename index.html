Emotional Speed Dating

Emotional Speed Dating is the result of a research project studying human emotion detection and machine learning. I created the study to the left which allowed me to understand how emotion detection works and thus enabled myself to create a game using Unity and C# that utilises emotion as an input. Within the game you see yourself participating in a speed dating event. The goal of the game is to find true love by empathising with your dates (mimicking their emotion). The game utilises a convolutional neural network as the main tool for emotion detection. You can read in full about the methods used in the study or have it summed up within the poster below.

Card Cavern

Card Cavern was created with Unity using C# in just under 12 weeks along side other projects. It is a rouge-like card game aimed to demonstrate my ability to bring a product to market in a given time frame. It was also created to show my knowledge of implementing monetization within applications shown by the in app purchases within the game. I also used this project as a way to practise my art skills and so all the art within the game was created by myself. One part of this project was to keep a sort of development log as I make the game. This can be seen below. The game is live on the google play store and the source code for the game is hosted on GitHub.

Unity Shader Tutorial

For one of my projects I was asked to create a tutorial on something I had learnt or would like to research more. I took this chance to delve deeper into the world of shaders within Unity. I created a C# tutorial to teach people the basics of the shader language to allow them to make their own shaders from the information gained in the tutorial. The tutorial has over 22,000 views with a 95.25% like to dislike ratio if that is any indication of the project's success.
