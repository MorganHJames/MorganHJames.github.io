Projects

Jerry's Quest (Unity/C#)

Jerry's Quest is the result of a group project. The project brief was to create a game suitable for a young age group. As the lead programmer and unity developer on this project, I learned the process and the necessity of good collaboration. The game was featured at ComX, a convention hosted by the University of Gloucestershire. It is a rare achievement for first year students work to be included as part of this convention. Here is the github for the project and here is a download for the unity files or you can just download the game.

Tiny Tanks (C++)

Tiny Tanks is a small game I made which takes advantage of a Math Library that I wrote. The Math Library contains collision functions along with vector and matrix classes amongst other helpful features. The game features 1 or 2 players, power ups, debuffs, independant turret/chasis rotation and shooting, of couse! Here is the github for the project and here is a download for the project files. Alternatively, you can just download the game or download the library.

Scientific Calculator (C++)

This scientific calculator implements reverse Polish notation and shunting yard algorithms. The calculator has all the buttons you would expect a normal scientific calculator to have, but also includes nine seperate memory modes to save different values. Here is the github for the project and here is a download for the project files. Alternatively, you can just download the executable.

Space Invaders (C++)

Space Invaders was one of the first fully fledged games I ever made. This version of the game was coded in C++ and features 3 difficulties, a local highscore, destructable bunkers and endless gameplay. Here is a download for the game.

Renesas MCU Car Rally (C++)


I was apart of a team of 5 students representing the University of Gloucestershire at the Renesas MCU Car Rally competition in Nuremberg, Germany in 2017. The competition is a time trial race against other universitys where we raced self driving cars that were controlled by microcontrollers. The cars were fabricated and programmed by the teams. As the project manager, I delegated jobs, tracked the progress and planed for the construction/testing of the car. I also contributed to the coding/fabrication of the car. Here is a link to the competition page.
