Jerry's Quest (Unity/C#)

Jerry's Quest is a collaboration between other students and I. We had to create a game following a game design document and suit it towards a certain age group. I was the lead programmer and unity developer on this project. The game was featured at ComX a convention hosted by the University of Gloucestershire. Here is the github for the project and here is a download for the unity files or you can just download the game.

Tiny Tanks (C++)

Tiny Tanks is a small game made to take advantage of a Math Library that I wrote containing Collision functions along with Vector and Matrix classes amongst other helpful features. The game features 1 or 2 players, Power ups, Debuffs, independant turret and chasis rotation and of course shooting! Here is the github for the project and here is a download for the project files alternatively you can just download the game or download the library.

Scientific Calculator (C++)

This Scientific Calculator makes use of a Reverse Polish Notation and shunting yard algorithms. The Calculator has all the buttons you would expect a normal Scientific Calculator to have but with the addition of 9 seperate memory modes to save differnt values to. Here is the github for the project and here is a download for the project files alternatively you can just download the executable.
